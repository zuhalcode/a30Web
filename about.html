Tentang Kami

A-03 adalah sebuah kelompok coding pada Bootcamp Progate x Universitas Negeri Surabaya. A-03 memiliki tugas membuat halaman web yang terdiri dari HTML, CSS yang responsif dan memiliki animasi. Maka dari itu kami membuat web ini sebagai project dari tugas kami juga sebagai portfolio kami. Kami berharap dengan membuat web ini dapa menambah ilmu, pengalaman, serta relasi.
